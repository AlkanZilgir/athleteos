# AthleteOS 🏋️

Your complete fitness PWA — works offline, no API keys, free AI included.

## Deploy to GitHub Pages (5 min)

1. Create a new **public** repo at github.com/new
2. Upload all 5 files: `index.html`, `sw.js`, `manifest.json`, `icon-192.png`, `icon-512.png`
3. Settings → Pages → Source: main branch → Save
4. Your app is live at `https://YOUR_USERNAME.github.io/REPO_NAME`

## Add to your phone

**iPhone (Safari):** Share ↑ → "Add to Home Screen"
**Android (Chrome):** ⋮ menu → "Add to Home Screen"

## Features
- ✅ Workout logging + PR tracker (week vs week)
- ✅ Protein, carbs, fat tracking
- ✅ Water tracker (tap cups)
- ✅ Meal suggestions (tap to add instantly)
- ✅ Food barcode scanner (Open Food Facts)
- ✅ Body weight chart
- ✅ Sleep tracker with quality rating
- ✅ Free AI trainer (Pollinations — no key, no cost)
- ✅ Daily personalized tip
- ✅ Workout streak counter
- ✅ Full offline support (Kosovo ✓)
- ✅ No API keys anywhere in the code
- ✅ Your data stays on your device only

## Offline (Kosovo / no internet)
Everything works offline except:
- AI chat (needs internet for Pollinations)
- Barcode scanner (needs Open Food Facts)
